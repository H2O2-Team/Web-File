# 模拟器对什么处理器兼容最好？

骁龙（SD）处理器因为有Turnip zink支持相较于麒麟和联发科（天机）使用virgl的模拟效率更高🙂

# 处理器要求

- 骁龙：建议在835及以上
- 联发科：天玑7200/8100/8200/9000
- 麒麟：麒麟8000

# 我该预留多少储存空间？

mobox很小的，只需要＜3G即可 但我们仍然建议您至少留出10G可用空间
